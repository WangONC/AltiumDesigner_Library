# Altium Designer Library

自制自用封装库，均由**Altium Designer 25.5.2** 版本制作，其余版本未经测试。

所有封装都根据数据手册由IPC Compliant Footprint Wizard生成并调整，**并非所有封装都经过实际的打板焊接测试**。

文件夹名称为元件供应商名称。
